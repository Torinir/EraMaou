eramaou モンスター召喚改良案

※(eramaou　ver.0.240推奨)

■使い方
　ERBファイルを上書きしてお使いください

■現状について
現在のモンスター召喚は１日の終わりに５種類がランダムで召喚され
1種類につき期待値は7匹、召喚数５回の合計は35匹の期待値となっています。
これは変化することがなく、日々の補給が足りない原因になっていると思います。

■改良案
そこで
魔王のレベルが１０、３０、５０、７０、１００になるごとに召喚回数が1回ずつプラスされる。
魔界知識を持っていればさらに召喚回数が+1回
レベル１００+魔界知識で合計12回を毎日召喚出来るようにする。

さらに
魔王のレベルが２０、４０、６０、８０、１００になるごとにモンスターの召喚数が１匹ずつプラスされる。
淫魔知識でさらに+1匹
つまりレベル最高のモンスターは１回の召喚で１か２匹しか召喚されなかったのが7か8匹召喚されることになります。

レベル１００以上+淫魔知識+魔界知識の召喚量の期待値は143匹となります。
これでレベル上げや知識習得のメリットが大きくなり、豊かなダンジョン生活を送れるようになれると思います。

■変更点
召喚回数となるREPEATの数値にローカル変数を使用し、レベルかTALENTに合わせて召喚回数が+1回追加されるようにする。
同じく召喚数もレベルかTALENTに合わせてモンスター召喚数が+1匹ずつ追加されるようにする。
