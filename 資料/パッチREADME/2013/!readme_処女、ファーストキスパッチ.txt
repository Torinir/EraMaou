■使い方
全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
ファーストキスの相手と初体験の相手の名前を保存出来るようにしました。
調教開始時メッセージに崩壊時の口上を追加。
結婚生活で妊娠するようにフラグを追加。

COMF_VAGINASEX.ERB

CHAR_MAKE.ERB

INFO.ERB

MARRIAGE_DAY.ERB
結婚生活で妊娠するようにしました。

SYSTEM.ERB
村娘追加時にファーストキス未経験フラグを追加。

SYSTEM_SOURCE_SUB1.ERB
処女を失った場合、初体験の相手をCSTR:3で記録するようにしました。
