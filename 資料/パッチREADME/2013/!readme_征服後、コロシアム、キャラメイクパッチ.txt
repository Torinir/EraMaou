eramaou_征服後、コロシアム、キャラメイクパッチ

このファイルはeramaou用です。

※(eramaou　ver.0.260推奨)

加筆・改変はご自由にどうぞ。

■使い方
eramaou_処女、ファーストキスパッチを適用してから
全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
・世界征服後にエルフの領域、ドラゴンの山、天界に侵攻出来ます。
　侵略度が10000を越えるとそれぞれエルフ、ドラゴン、天使の勇者が1人だけ貰えます。

・コロシアムでのバグ修正とファーストキスと初体験の相手を記録します。

・結婚式の時にファーストキスと初体験の相手を記録します。


COMF31.ERB
コロシアムでのフェラチオ時にファーストキスの相手を記録します。

COMF204.ERB
COMF205.ERB
COMF206.ERB
バグ修正とトロルに犯される時に拡張経験＆異常経験（未拡張時１回のみ）がつくようになりました。

DUNGEON_RYOUZYOKU.ERB
ダンジョン内での陵辱時にファーストキスと初体験の相手を記録します。

SYSTEM_SOURCE_SUB1.ERB
コロシアムでの初体験の相手を記録します。

INVASION.ERB
エルフの領域、ドラゴンの山、天界を追加しました。
世界征服後に行けます。
追加フラグはフラグまとめに追記しました。

INVASION_RYOUZYOKU.ERB
エルフの領域、ドラゴンの山、天界に対応するようにしました。

INFO.ERB
ファーストキスと初体験の相手を追加しました。

MARRIAGE.ERB
結婚時にファーストキスと初体験の相手を記録します。

SYSTEM.ERB
バグ修正とあなたのファーストキス未経験を設定。

ENDING.ERB
エルフの領域、ドラゴンの山、天界のエンドを追加しました。
追加フラグはフラグまとめに追記しました。

LOOK.ERB
髪の色にエルフの領域、天界、ドラゴンの山のクリアフラグをつけました。
ドワーフとホビットの処理をCHAR_MAKE.ERBから移動しました。

CHAR_MAKE.ERB
キャラ生成時の素質の付き方について色々と変更しました。
