eramaou_人数制限緩和 20131026

※(eramaou　ver.0.270推奨)

加筆・改変・再アップロードはご自由にどうぞ。

■使い方
　eramaou_0.270パッチまとめ20131019を適用してから
　全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
2013/10/26
・エルフの領域、ドラゴンの山、天界を征服した時にキャラ保持人数が多すぎてもキャラが貰えるように変更。
・新しい勇者がダンジョン攻略するときに、日数が増えると最大30人まで増えるように変更。

■ENDING.ERB
各領域にあった以下の文を削除
IF CHARANUM > 20
	PRINTW ＊キャラが多すぎます、減らして下さい＊
	RETURN 0
ENDIF

■ENTER_ENEMY.ERB
23行以降の「キャラが多すぎる場合中断」に追記。
最大30人（あなたも含むので勇者は29人）まで増えます。
