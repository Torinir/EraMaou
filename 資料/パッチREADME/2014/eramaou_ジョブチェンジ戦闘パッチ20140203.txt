・転職時にパラメータを再設定（レベル１よりは強くなります）

■修正箇所

　IF FLAG:5 & 32
　	CALL ATTACK_KOUJO
　ENDIF
　を追加。

　■DUNGEON_BATLLE2.ERB
　CALL ATTACK_KOUJO**の間に

　IF FLAG:5 & 32
　	CALL ATTACK_KOUJO**
　ENDIF
　を追加。
