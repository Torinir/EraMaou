■使い方
　全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
・コロシアムでオトコ素質を持っていても膣を使用出来ないようにしました。
・常時発情時のバグを修正。
・外界侵攻によるオーバーフロー問題とその解決方法の試案.txtを参考に侵攻時のバランス調整。
・モンスターの戦力を下方修正。
・侵攻時の表記を修正（"あなたの気力"と表示）

■追加・修正箇所
　■COMF201.ERB
　各コマンドに以下の文を追加。
　SIF (TALENT:ASSI:121 == 1 || TALENT:ASSI:122 == 1 || ITEM:PBAND == 1) && TALENT:122 == 0
　	PRINTL [2] - 膣
　～～～～～
　ELSEIF RESULT == 2
　;対象がオトコなら戻る
	SIF TALENT:122
	RETURN 0

　■COMF202.ERB～COMF208.ERB
　各コマンドに以下の文を追加。
　SIF TALENT:122 == 0
	PRINTL [2] - 膣
　～～～～～
　ELSEIF RESULT == 2
　;対象がオトコなら戻る
	SIF TALENT:122
	RETURN 0
