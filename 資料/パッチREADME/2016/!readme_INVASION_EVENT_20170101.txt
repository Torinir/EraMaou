eramaou_INVASION_EVENT_20170101

eramaou ver.0.800用
侵略イベントパッチ

※加筆・改変・再アップロードはご自由にどうぞ。

2017/1/1
■使い方
　全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
・侵攻度によってイベントが起こるようにしました。
・人間界で侵攻度が2000、4000、6000、8000、10000で侵攻イベントが開始されます。
・ただし中身はほぼありません、いっぱい加筆してくだち…
・雛形としてエルフ、ドラゴン、天界の領域も用意しました。

■追加・修正箇所
　■INVASION.ERB
　　914行目から@INVASION_EVENT関数に飛ぶようにしました。

　■INVASION_EVENT.ERB
　　侵略イベント関数です。

　■新しく使用するフラグは以下の通りです
　　FLAG:93 = 人間界の侵攻イベントフラグ
　　FLAG:94 = エルフの領域の侵攻イベントフラグ
　　FLAG:95 = ドラゴンの山の侵攻イベントフラグ
　　FLAG:96 = 天界の侵攻イベントフラグ
