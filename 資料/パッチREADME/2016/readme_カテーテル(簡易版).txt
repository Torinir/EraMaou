eramaou_カテーテル追加パッチ

※(eramaou　ver.0.750推奨)

加筆修正・改変・再アップロードはご自由にどうぞ。

■使い方
　全てのERBファイルをERBフォルダに上書きコピーしてください。

■仕様
・その他の装備にカテーテルを追加し購入できるようにします。
　┗「神秘の尿道カテーテル」を追加。
    ※購入に3000ポイント、従順3LV以上必要です。

・装備するとどんな事が起こるの？
　┗日を跨いだ際のおねしょイベントで衣服が汚れません。
    魔王部屋にて調教中・待機中の際は文章や効果が出ます。
    (簡易版なので、迎撃中、進行中などの他の状態時は、何も文章は無いです。
     ただ、おねしょイベントによる衣服の汚れは回避します。ダンジョン内での快楽刺激などによるおもらしは対処できてません)
　┗蛇足かもしれませんが、購入して装着する際、ちょっとした文章が出ます。不評でしたら消します。

・外してて持ってるだけでおねしょ回避するんだけど...
　┗装備品の特性、特別コスチューム、の仕様上の都合みたいで、今の所対処法見付かってないです。
    そもそも、調教の際にわざわざ外す、という事を行うメリットゼロですけど...

・「FUNC_CLOTH.ERB」の中身見たら、更になにかあるのですが？
　┗まだ作製途中です。それと試作のつもりで作ってた番号のまま装備品の内部使用番号(CFLAG:42)使ってるので、
    maou作者様からなにか言われたら変えるつもりです。
    試作時の際、自分用にわかりやすい離れた番号使ってますが、間を空けたの不味かったかなあ...


■追加・修正箇所
　■EVENT_NEXTDAY.ERB
　　「【おもらし癖】持ちのおねしょチェック」にて追記。

　■FUNC_CLOTH.ERB
　　装備品の追加。

　■SHOP_TAILOR.ERB
　　その他の装備品で「神秘の尿道カテーテル」が購入できるようになりました。
    購入からの装着の際、ちょっとした文章が出ます。

■詳しい追加内容
・EVENT_NEXTDAY.ERB
  おねしょイベントの際カテーテルを装備しており、魔王部屋にて調教中または待機中の場合、カテーテルのイベントが起こります。
  イベント分岐は従順LVにて起こります。その後の一部の分岐では、所持してる素質で更に変化します。
  男女どちらでも違和感無い様に書きました。

・FUNC_CLOTH.ERB
  装備品の追加。

・SHOP_TAILOR.ERB
  カテーテルが購入できるようになり、購入時における装着時には反応の様な文章が出ます。


■Q&Aっぽいもの
Q.どうして作ったの？
A.ちょっとしたイベント確認とかで日を送ってる際、衣服汚れたり、オムツ変更の選択が出ない様にしたかった。
  衣装云々も取り替えたりするの割と手間ですし、ポイントもかかる。作者様への負担も気がかりです。
  それと、恥らう女の子の反応が出るようにしたかった。

Q.なんで「神秘の」なの？
A.ランダムに作られた勇者にランダムで装着させたい人が出てくるかもしれなかったため、「魔の」というのは使いませんでした。
  また、侵攻して来る勇者にも反応、描写を描きたく、その勇者が禍々しい装備品普通に使ってたら変かな？ と思いました。

Q1.実際のカテーテルと大分違うような...？
Q2.なんか珠がもらえた。
Q3.もらした筈のものはいったいどこへ？
Q4.カテーテルが伸びる？
Q5.放尿経験数増えないの？
など言われそうな事について、
A.一言で述べるなら、「ファンタジーっていいね」になります。それぞれ書きますと、
  A1.実際の仕様から逸れないようにしてしまうと、行いたい事の制約になりかねないためです。
     そもそもそれほど詳しいわけでもないので、ほぼ適当な想像によって書いています。
  A2.珠がもらえる事については、「日頃から着ていられる調教器具」という機能も実装したかった為です。
     マイナスとなる否定の珠は、増やさないつもりです。
  A3.今のところ、無意識に汚れるのを嫌った勇者にカテーテルが反応して、カテーテルの先から出てしまうものを他の場所へ転移させた。
     または、
     魔王が余計な手間が増えないよう指定の場所へ転移させるようにしている。
     という設定のつもりです。魔王なので神秘の道具の力に介入してそれ位できても不思議ではないかな、と。神秘の力もすごい。
  A4.神秘の力を秘めているから、ある程度伸びるくらいなら別に変じゃないと思いたい。
     なので、体にフィットする事が起こっても不思議じゃない。
     そもそも貞操帯装着時に、種族や性別変更してもなんともないし...
  A5.おもらしイベントの発生条件が、放尿経験に依存しているようで、装着してても毎回増えてしまうと、
     日付け変更の際、毎回イベントが発生してしまい、長い文章の表示になるのは避けた方がいいかな、と思いました。
     また、意思とか関係なく出しっぱなしとして考えられたら、それ経験数になるのかなあ、と。
  こんな具合で、ゆるふわ設定な装備品になってます。

Q.「簡易版」？
A.色々書いたのですが、状態によってはおもらしイベントというより、「日頃から着ていられる調教器具」としての反応しか書けてない部分があり、
  その反応で状態事のイベントが起こると「ちょっと変かな？」、と思ったので書き直してます。
  例えば苗床の状態なのに、おねしょイベントで文章の内容が「日頃から着ていられる調教器具」としての反応っぽい。
  などです。いろいろと書き直してます。



■追加予定
・おねしょイベントだけでなく、「日頃から着ていられる調教器具」としての効果、
  調教時も含む放尿におけるカテーテルを装着してる場合の反応、衣服汚れの回避も書きたい。

・貞操帯機能を含むもの(神秘の尿道カテーテル付き貞操帯)も作りたい。ただいま製作中...

・男女それぞれの反応、描写が思いついたら、書くかも？
  男のはそもそも考えてない（現状書く気がない）、ので（要望があり）思いついたら分岐条件含めて書くかも。
  現状としては、性別男の場合、特に表示無しというのでいいかな、とも思う。

・余力があれば反応や描写の際、性格やユニークキャラ事に一言台詞が出るようにするかも。

・あとは何か思いついたら書くかも。



■追加したい事(カテーテル関係以外で。とりあえず提案)
・「亀甲縛り用の縄」に装着して日付変わる際、なにかしら効果書いてみたいかも。余力があれば。


■もっと追加したい(案だけ)
・階層毎の施設で、「氷室」「沼地」とかありますけど、カテーテル装着したまま行ったら、
  すごい大変な事になりそう、とreadme書いてて気付いた。それと罠の「媚薬沼」とかも。
  凍ったり、媚薬が体内に直接流れてきそう。
